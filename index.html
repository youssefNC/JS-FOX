تلخيصات الدورات التعليمية الخاصة بموقع اكاديمية الزيرو

مرحبا بك في موقع تلخيصات الدورات التعليمية. يعتبر الموقع مرجعا لبعض دورات اكاديمية الزيرو, حيت تستطيع الرجوع للملخصات للمراجعة.

ما هي لغة جافا سكريبت

لغة الجافا سكريبت هي لغة برمجة Programming Language او Scripting Language تعمل على جانب ال Client Side او Server Side. تُستخدم الجافا سكريبت لجعل صفحات الويب تفاعلية.

ما هي الاشياء التي يمكن ان نفعلها بالجافا سكريبت؟

  • تحديث المحتوى ديناميكيا
  • التلاعب بعناصر ال HTML و CSS
  • تحريك المحتوى والصور مثال: slider
  • التلاعب والتحقق من صحة البيانات
  • التحكم في الوسائط Audio, Video
  • إنشاء الالعاب على المتصفح
  • إنشاء تطبيقات الهواتف وسطح المكتب

بعض الإضافات المهمة داخل محرر الاكواد visual studio code

قم بالضغط على ايقونة extensions لتتبيت الاضافات

بعض الاضافات التي سوف تساعدك في مشاريعك البرمجية

Bracket Pair Colorizer 2

هذه الإضافة تقوم بتلوين الاقواس بألوان مختلفة عند كتابة كود متداخل حيت تمكننا من التمييز بين بداية القوس ونهايته.

EditorConfig

هذه الإضافة مفيدة عند العمل مع فريق حيث تسمح لك بانشاء ملف config للاعدادت الخاصة بك على محرر الأكواد. عندما يقوم شخص اخر في الفريق بتنزيل الملف فإنه يحصل على نفس الإعدادات.

ESLint

هذه الإضافة تساعدك في كتابة كود سليم متوافق مع المعايير العالمية لكتابة الكود.

indent-rainbow

هذه الإضافة تقوم بتلوين المسافات البادئة أمام الكود الخاص بك بالتناوب بأربعة ألوان مختلفة في كل سطر.

Live Server

تمكنك هذه الإضافة من انشاء خادم محلي local Server لتجربة البرمجيات الخاصة بك. وتتوفر ايضا على خاصية live reload التي تقوم باعادة تحميل الصفحة تلقائيا عند حفظ المشروع.

Material Icon Theme

هذه الإضافة تقوم بإضافة أيقونات لمحرر الأكواد حيث تضيف شكلا جماليا للمحرر

Path Intellisense

هذه الاضافة تقوم بإكمال أسماء الملفات تلقائيا عند كتابة مسار معين, وتسمح لنا أيضا بإستكشاف الملفات والإختيار منها.

Prettier - Code formatter

هذه الإضافة تقوم بتنسيق الكود الخاص بك
